# Retail Store Discounts(Using SpringBoot with Kotlin)

## Requirements
JDK -11
Kotlin -1.5.20

### Used Maven for Building & for packaging jar files.

## How to Run the Project

Go inside the Retail-Store directory and run the command (mvn spring-boot:run)
